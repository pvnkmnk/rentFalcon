üîß Troubleshooting

Problem: "Python is not installed"

  • Install Python from python.org
  • During installation, CHECK the box: "Add Python to PATH"
  • Restart your computer
  • Run EASY_SETUP.bat again

Problem: "Chrome not found"

  • Install Chrome from google.com/chrome
  • Run EASY_SETUP.bat again

Problem: No results found

  • Check spelling of city name
  • Remove price filters (leave blank)
  • Try a different city (Aurora, Richmond Hill, etc.)
  • Check internet connection

Problem: First search is very slow

  • This is normal! ChromeDriver is downloading (30-60 seconds)
  • Only happens once
  • All subsequent searches are fast (10-15 seconds)
