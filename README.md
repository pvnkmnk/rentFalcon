# Rental Scanner

This application scans Kijiji for local rental and lease postings based on user-defined criteria.

## Features (Planned)

- Price Range filtering
- Location filtering
- Scheduled scans and reports
- Tracking of removed listings

## Setup

1. Create a Python virtual environment: `python -m venv venv`
2. Activate the virtual environment: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Linux/macOS)
3. Install dependencies: `pip install -r requirements.txt`
4. Run the application: `python app.py`
